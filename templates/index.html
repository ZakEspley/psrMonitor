PSR Monitor Controller

Change what is displayed to the students

Use this website to control what is displayed on the PSR Monitors. The main steps for making a slide show is the the following

  1. TA's and LA's are referred to as "hosts". Add all the hosts for the quarter by uploading a CSV file that matches the template shown in teh "Add Hosts" page.
  2. Once all the accounts for the hosts have been added, you need to upload a picture for each host. To do this, you should have a picture of the host that contains only the one individual with a little bit of space around their face. The file name should be the host's email address you used when making the profile. The images will be cropped to a 500px square around their face.
  3. Verify that the images worked out. You might find that the algorithm did a bad job cropping, didn't find a face, found too many faces, or couldn't find an account. Fix any problems that occurred by manually uploading photos for the hosts.
  4. Create the slide show. To do this, you will need to upload a CSV file with all of the times and people in the each timeslot for the PSR.
  5. Add any advertising slides. If the students or department wants to advertise an upcoming event you can add it in teh "Add Slides" page.
